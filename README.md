
## 项目名称

Lazarus IDE移植

## 支持单位

龙芯中科技术股份有限公司

## 项目描述

lazarus是一款fpc语言的编程环境，使用fpc语言编写。目前LoongArch已经支持fpc 3.3.1的中间版本。项目目标是适配新的编程环境，移植lazarus使其在LoongArch平台可用。
